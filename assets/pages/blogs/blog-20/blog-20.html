How To Install And Configure PhpMyAdmin Ubuntu 20.04 Linux.


1. Step - Update Server

Update your package lists from the repositories. To do this, open a terminal. Now type the following commands in the terminals.

sudo apt-get update

sudo apt-get upgrade


2. Step - Install nano, mysql server and phpmyadmin

Now install nano with the following command:

sudo apt-get install nano

sudo apt-get install mysql-server

sudo apt-get install phpmyadmin apache2-utils


3. Step - Configuration of phpMyAdmin

The images I show here are automatically displayed when you run the installation process in the terminal.


The first question is, which web server do you want to use? We are going to use Apache2 so: Press the spacebar then press the tab button and then press enter.


Question two is about how we want to configure the MySQL server. We just click enter, then this configuration process is done automatically, and we don't have to set it up manually.


Question three. You must provide a password for the application in question, so type the password click on tab and then press enter.


Question four, they want you to repeat your password, so type the password click on tab and then press enter.

4. Step - Configuration of MySQL server

We need to connect to the SQL server. We do that with the following command. The default user is root without a password.

sudo mysql -u root



Now we will create a new user. You do this with the following command. Make sure you change user with your username and password with your password.

CREATE USER 'user'@'localhost' IDENTIFIED BY 'password';



Now we will give the necessary permissions. To the new user.

GRANT ALL PRIVILEGES ON * . * TO 'user'@'localhost';



And then exit MySQL server interface. (Type in exit.)

5. Step 5 - Login Into the web interface

Type in your IP address and then the following path.

/phpmyadmin/index.php


Example.

http://192.168.1.140/phpmyadmin/index.php


Then you will get a web interface. Type in your username and password.


Then press go and you're ready to go.
