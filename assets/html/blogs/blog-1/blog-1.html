1. Domain Name System (DNS)
1.1. Configuration of the network card

You can turn a Windows 10 server 2019 into a DNS server by installing the DNS service.

This server must have a static IP address. You can do this by press the Windows key and type "ncpa.cpl".

Disable IPv6 protocol

IP-adres = 192.168.10.100

Subnet mask = 255.255.255.0

Default getway 192.168.10.254

Preferred DNS server = 127.0.0.1

Restart your server after giving a static IP address!!!

1.2. Installing the DNS service

Go to "Server Manager Control".

Click on "Add roles and features".

Click on "Next".

Click on "Next".

Click on "Next".

Left click on the square next to "DNS Server".

Then click on "Next >".

Click on "Add Features".

Click on "Next".

Click on "Next".

Left click on the square next to "Restart the destination server automatically if required".

Click on "Yes" for an automatic restart of the server.

Finally click on "Install" (The server will be occupied for a few moments to process the installation.)

Click "Close" if the installation is successful.

1.3. Set DNS suffix name for the server

Press the Windows Key and type "sysdm.cpl".

Click on "Change".

Type in the computer name field "serverelias" and then click on "More".

Enter a DNS suffix, this is actually a domain name. This one for me is "school10.local" Click "OK" click "OK" again.

Click "OK" to restart the server Then press "Close" and "Restart Now".

1.4. Creating a forward lookup zone in the DNS console

In your "Server Manager Control" you will see "Tools" at the top right. click on it and select from the list DNS.

Click on your server name (serverelias).

Double click on "Forward Lookup Zones".

Right click on "Forward Lookup Zone" and select "New Zone...".

Click on "Next".

We choose "Primary zone" and click on "Next".

Enter the name of your zone here, which must match your "DNS suffix name" of your server. Then click on "Next".

We want to create a new zone database file, so just click "Next". A new zone database file is created. click on "Next".

Click on "Finish".

1.5. Setting up a forwarder

Open your DNS console.

Right click on your server and select "Properties".

Go to the "Forwarders" tab and click "Edit". Enter the following IP address here: 8.8.8.8 If you have internet, the IP address will be resolved in dns.google

And click on "Interfaces" Select "Only the following IP addresses". And click on "OK".

1.6. Disable Internet Explorer Enhanced Security Configuration

In "Server Manager Control" go to your local server. Click on "On".

Click "Off" under Administrators below.
