How to install windows server 2022 on PowerEdge R710



1. Step one

Plug in the power to the server.

Press F2

Press enter on "Boot Settings"

Press up on "Boot Mode" to change BIOS to UEFI and then press esc.

2. Step two

Press Ctrl + R

Press F2

Press on "Create New VD"

Add the disks and name the raid

And then put a boot USB in the server + reboot
