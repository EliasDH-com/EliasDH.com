Windows Server 2019 Permissions Setup



1. File and folder permissions

1.1 Exercise one permissions

In this exercise we create a folder "student_data" in the root of the partition C:\ in

this folder all students have their own working folder in which they can create and

delete new files and folders. The "student_data" folder and their own workbook

cannot be deleted by the students. The subfolders and files in their own

workbook may be deleted by the students. The administrator does this job by

granting the appropriate permissions. The administrator and System have Full

Control on all folders.

Create the student data folder with the 5 subfolders.

Let"s verify the "Security" tab of the C:\ partition. We have 2 groups that have full control over the C:\ partition. Click "OK".

Go to the "Security" tab of the student data folder and click "Edit".

Click the "Add" button.

Type in "cursisten" and click "OK".

The students have joined, but do not yet have the appropriate permissions. Click "OK".

Select students and click "Advanced".

Select students and click "Edit".

Click on "Clear all" and then click on "Show advanced permissions".

Applications to = This folder only

Put check marks next to "Traverse Folder / Execute File", "List Folder /Read Data" and "Read attributes" then click "OK" twice.

Now set the following default permissions for "Adrie Beke" for the folder "Beke". He got the first three by default, Write permissions, you still have to put a check there.

Now do the same for the other four students. In their own folder of course.

Of course we will have to share the "student data" folder. Go to the Properties of the student data folder.

Go to the Sharing tab and click on "Advanced sharing". Activate "Share this folder".

Click on "Permissions".

Remove "Everyone" and add the "students" and "administrators". Give the 2 groups full control.

Go to the Properties of the "Adrie" folder and then to the Security tab. Then go to the Advanced Permissions and check the box next to "Delete Subfolders and Files".

Sign up under "Adrie" and check!!
