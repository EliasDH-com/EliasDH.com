Configuratie

Web Server Configuratie

Wat betekent dit concreet.

De server (Raspberry Pi) is geconfigureerd met Apache2.

Apache is --Apache HTTP Server is een opensourcewebserver voor Windows, macOS, Linux en andere Unix-achtige besturingssystemen.

Apache HTTP Server wordt ook wel Apache of httpd genoemd. Oorspronkelijk was Apache het enige opensource alternatief voor de Netscape-webserver.--

Wikipedia

Wat heb ik gedaan met Apache, ik heb Apache gedownload op de server (Raspberry Pi) met sudo apt-get install apache2.

En dan heb ik een .HTML file gemaakt en geplaatst in /var/www/root/web-server

En ook nog twee Aparte configuratie file aangepast voor de nieuwe .HTML pagina te activeren, in /etc/apache2/sites-enabled en /etc/apache2/sites-available

En alles laatsten de web server is beveiligt met een ssl (Transport Layer Security) certificaat (met een Script genaamd letsencrypt).

Wat heb ik gedaan met MariaDB, ik heb MariaDB gedownload op de Server (Raspberry Pi) met sudo apt-get install MariaDB.

En dan heb ik een dadabase gemaakt genaamd cloudserver.

En de dadabase staat in /var/lib/mysql/cloudserver

En de front end files staan in /var/www/root/cloud-server
