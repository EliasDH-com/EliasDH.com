1.2 Using Print and Document Services (PDS)

Go to Print Management via Tools.

We see that our "OKI" is listed at the printers tab.

Right click on "OKI" and select "List in Directory".

Click on "Managing Sharing".

Go to the Security tab.

Remove “Everyone” by clicking the “Remove” button.

Make sure that only "Administrator", "Administrators", "Server Operators" and "Print Operators" remain.

Click on "Add".

Type in "cursisten" and click on Check Names.

Students appear, click OK.

Students only get Print permissions. Click OK.

5.3 Install Printer Driver on Client Computer

Log in to the workstation.

Search \\serverelias

Click "Install Driver"

If requested, also provide the username and password of the person who is authorized to install the printer.

Right click on the white area.

Select Properties.

Click Print Test Page.
