Windows Server 2019 Group Setup

1. Groups

1.1 Example use of groups

We have 7 domain users. The first five are students.

The last 2 are secretaries and work in the administration.

Usernames Log in as
Rene Corneels corneels@school10.local
Adrie Beke beke@school10.local
Arie Stroop stroop@school10.local
Veerle Driessens driessens@school10.local
Bieke Jaspers jaspers@school10.local
Bea Tram tram@school10.local
Anna Trapvod trapvod@school10.local

Create these 7 users. We will create 2 global groups here with the following names: students and secretaries.

Create the 7 domain users.

Create a new group via "Users" -> "New" -> "Group"

First we create a group of students. This is a global domain group of type Security.

Make the five students members of this group.

Next, create a group of secretaries. This is also a global domain group of type Security.

Make Trapvod and Tram a member of this group.
