Windows Server 2019 Home-Folder Setup

1. Home-Folder

1.1 Create home-folder

In the C:\ partition we create a folder called home-folder.

This folder must be visible to the users over the network, so share this folder (the name of the share is the same as the name of the folder).

Go to "Properties" and go to "Sharing" tab, click on "Advanced Sharing" button.

We make sure that we only have Share Permissions for the "Domain Users" and the "administrators". We give these "Full control". We remove "Everyone" from the share.

In the "Security" tab were the "Administrators" and the "System" with "Full Control". Let"s leave this alone.

1.2 Assigning a home-folder to users

We had created 5 students. We will give the students a home-folder.

Go to ADUC and double click on "Adrie Beke". Select the "Profile" tab.

In the field "Home folder" select the radio button "Connect" and select Z. In the field "To" type in \\serverpaul\homefolder\%username% and click on "Apply". You will now see that %username% is changed to beke. %username% is called a variable. Click "OK".

We now see in explorer that a "beke" folder has been created in the homefolder.
