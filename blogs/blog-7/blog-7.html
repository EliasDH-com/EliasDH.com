Windows Server 2019 Profiles Setup

1. Profiles

1.1 Create Roaming Profiles

Create and share a folder to store roaming profiles. Create a profiles folder on the C:\ partition and share this folder. "Administrators" and "Domain users" get "Full control". The user "Everyone" must be deleted.

In the tab "Security" we see the "Administrators" and "System" with "Full Control". This is "OK". Now also add the "Domain Users" group to it and give it "Full Control".

Modify account of beke to use roaming profile. Go to ADUC and double click on "beke". Select the "Profile" tab. In the "Profile Path" field, type the following in \\serverpaul\profiles\%username%. Click on "Apply".

%username% is changed to "beke". Click "OK".

Have a look in the profiles folder, it is still empty.

Now log in as "beke" on a workstation. Now look in the "profiles" folder on the server. The system has created a folder "beke.V6" to store beke"s profile. By default, the profile on the users server is not accessible to the administrator.
